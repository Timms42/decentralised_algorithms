# Code for numerical experiments
## For details, see [Tam, Timms, and Zhang, _ArXiv preprint_, 2025](https://arxiv.org/abs/2512.12502)

* algorithms.py - code for Algorithm 1 in [Malitksy and Tam, 2026](https://arxiv.org/abs/2308.11876) and Algorithms 1 and 2 in Tam, Timms, and Zhang (2025)
* game_matrix.ipynb - code for zero-sum matrix game numerical experiment in Section 4.1
  * requires algorithms.py
* game_vpp.py - code for virtual power plant model numerical experiment in Section 6
  * requires algorithms.py and vpp_data (contains dataset)
* robust_ls.ipynb - code for robust least squares numerical experiment in Section 4.2
  * requires algorithms.py and rls_synthetic (contains dataset)
